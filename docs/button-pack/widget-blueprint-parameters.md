
---

**All variables have Instance Editable and Expose on Spawn set to True**

Configure them per widget by modifying the associated variable in the widget instance or swap defaults by changing the variable default in the variable details panel.

## WBP CircleButton
| Variable Name | Type        | Category   | Description                                                     | Default Value        |
| ------------- | ----------- | ---------- |-----------------------------------------------------------------|----------------------|
| Button Icon   | Slate Brush | Appearance | The Image for the icon of the button.                           | X Icon Blue and Gold |
| Button Image  | Slate Brush | Appearance | The button image. This should generally be a material instance. | Slate Brush          |
