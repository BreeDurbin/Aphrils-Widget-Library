
---

**All variables are Instance Editable and set to Expose on Spawn.**

Configure each widget by modifying its instance variables, or change the default values in the Variable Details panel.

## WBP CircleButton
| Variable Name | Type        | Category   | Description                                                             | Default Value               |
| ------------- | ----------- | ---------- | ----------------------------------------------------------------------- | --------------------------- |
| Button Icon   | Slate Brush | Appearance | The image displayed as the button icon.                                 | X Icon Blue and Gold        |
| Button Image  | Slate Brush | Appearance | The image for the main button. Should generally be a material instance. | MI CircleButton BlueAndGold |
